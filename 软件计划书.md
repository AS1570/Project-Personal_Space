# Personal Space（个人空间）软件计划书

**版本：1.0.0** &nbsp;|&nbsp; **日期：2026-06-28**

---

## 一、项目概述

Personal Space 是一款基于 **WPF（Windows Presentation Foundation，.NET 10.0）** 开发的桌面应用程序。软件的核心定位是：为用户提供一个安全、私密、高度可定制的「个人专属数字空间」，并内置简易的伪装/隐藏机制以保护用户隐私。

用户可以在此空间中管理文件、撰写日记、播放音视频、浏览图片、自定义桌面布局，所有数据通过 **SQLCipher 加密存储**，通过密码锁屏保护。此外，软件支持一键伪装模式，可更改窗口标题和图标以伪装成其他应用程序（如计算器、记事本等），有效保护用户的使用痕迹和隐私。

## 二、问题描述与设计目的

### 2.1 问题描述

在共享电脑或办公环境中，用户常常面临以下问题：

| 编号 | 问题 | 说明 |
|:---:|------|------|
| 1 | 隐私数据缺乏保护 | 个人文件、日记、图片、视频等敏感内容容易被他人在文件资源管理器中直接发现和访问。即使使用操作系统层级的账户保护，在用户离开座位而未锁定屏幕时，隐私数据仍然暴露无遗。 |
| 2 | 缺乏统一的个人空间 | 个人文件（文档、图片、视频、音频、日记）分散在不同目录，缺乏一个统一的、一体化的管理入口和浏览体验。 |
| 3 | 使用的「隐蔽性」需求 | 在某些场景下，用户不仅需要保护数据内容本身，还需要掩盖「正在使用隐私软件」这一事实。主流的密码保护软件往往有明显的名称和图标（如"保险箱"、"加密空间"等），容易引起他人注意甚至怀疑。 |
| 4 | 个性化需求未被满足 | 大多数文件管理或加密工具缺乏灵活的可定制性，用户无法按照自己的审美和习惯布置个人空间。 |

### 2.2 设计目的

本软件旨在解决以上问题，具体目标包括：

1. 提供一个密码锁屏保护的安全入口，所有核心数据通过 SQLCipher 加密存储在本地数据库中。
2. 将文件管理、日记记录、音视频播放、图片浏览等功能整合在一个统一的、美观的桌面界面中。
3. 提供「伪装模式」——一键切换应用程序的窗口标题和图标，使其在外观上伪装成普通办公软件或系统工具，消除他人对"私密软件"的猜疑。
4. 提供高度自由的可定制性：主题颜色（浅色/深色/跟随系统）、强调色、壁纸（纯色/图片/视频）、顶部栏/底部栏的行为模式、桌面小组件系统等，每个用户都可以打造完全属于自己的个性化空间。

## 四、已实现功能详细说明

### 4.1 密码锁屏系统（UserControl0）

- 启动时展示锁屏界面，显示实时数字时钟和日期。
- 点击界面任意位置进入密码输入面板。
- 使用 SQLCipher 加密的 SQLite 数据库验证密码。
- 支持密码提示词（可修改）。
- 首次错误输入后显示密码提示。
- 锁屏壁纸支持：无 / 纯色 / 图片。
- 时钟和日期文字颜色可分别自定义。
- 密码可在设置面板中修改。

### 4.2 个性化桌面（UserControl1）

**(1) 桌面壁纸**

- 支持四种模式：无壁纸 / 纯色填充 / 静态图片 / 视频动态壁纸。
- 图片壁纸支持多种格式（PNG、JPG、BMP、GIF、WEBP）。
- 视频壁纸循环播放且自动静音。

**(2) 顶部栏（TopBar）**

- 三种显示模式：展开（Docked，始终显示并限位）/ 悬浮（Floating，显示但不限位）/ 自动隐藏（AutoHide，鼠标靠近时显示）。
- 实时时钟和日期（格式高度可定制）。
- 窗口控制按钮（最小化、最大化/还原、关闭）。
- 可选的"锁定"按钮（点击返回锁屏）。

**(3) 底部栏（BottomBar）**

- 两种视觉风格：悬浮式（Floating）/ 底部联通式（Connected）。
- 三种按钮布局：居中（Center）/ 左右两侧（Full）/ 中心+右侧（CenterRight）。
- 三种可见模式：展开 / 悬浮 / 自动隐藏。
- 10 个功能按钮（可选择性显示/隐藏）：

  | 按钮 | 功能 | 图标 |
  |:---:|------|:---:|
  | Button 1 | 显示桌面（返回桌面） | 🏠 |

  | Button 3 | 音频播放器 | 🎵 |

- 播放/暂停、上一首/下一首。
- 进度条拖拽。
- 音量控制滑块。
- 循环模式切换（顺序循环 / 单曲循环）。
- 快速选择音频文件（支持单曲/列表模式）。

- 禁用伪装后自动恢复原始标题和图标。
- **任务栏中显示的窗口名称和图标也同步变化**，达到以假乱真的效果。

## 五、数据安全架构

| 安全要素 | 实现方案 |
|----------|----------|
| 数据库文件 | `appdata/data.db`（SQLCipher 加密） |
| 默认密码 | `admin123`（首次启动自动创建加密数据库） |
| 密码修改 | 通过 SQLCipher `PRAGMA rekey` 实现安全的密钥变更 |
| 用户文件 | 存储在 `ProgramFile` 目录（路径可自定义） |
| 会话安全 | 数据库在锁屏状态下自动断开连接，密码验证通过后才重新连接 |
| 敏感设置 | 所有敏感设置（密码提示词等）存储在加密数据库的 `EncryptedSetting` 表中 |
| 日记数据 | 日记内容存储在加密数据库的 `Diary` 表中 |

## 六、技术亮点

1. **双资源字典主题系统**
   Dictionary1.xaml（浅色）和 Dictionary2.xaml（深色）通过 ThemeService 动态切换，支持手动/定时/跟随系统三种模式。

2. **SQLCipher 加密数据库**
   所有隐私数据（日记、加密设置）均加密存储，即使数据库文件被复制也无法在没有密码的情况下读取。

3. **网格小组件系统**
   完整实现了类似 Windows 磁贴的网格对齐系统，支持拖拽、缩放、吸附、比例式缩放、Z 轴管理等。

4. **伪装模式**
   一键改变应用程序外观身份，在操作系统任务栏层面实现"隐身"。

5. **多媒体集成**
   基于 WPF MediaPlayer/MediaElement 实现音频和视频播放，支持循环模式、进度跳转、音量控制等完整功能。

6. **文本编辑器高级特性**
   100 级撤销/重做、多编码支持（含中文 GB2312）、查找替换、滚轮缩放等。
